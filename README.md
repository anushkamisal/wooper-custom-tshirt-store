# Wooper Custom T-Shirt Store

A custom T-shirt store website where users can browse, search, and customize T-shirts with different colors, sizes, prints, and designs.

## Features
- Browse different T-shirt collections
- Search and filter products
- View product details
- Select size, color, and quantity
- Add products to cart
- Wishlist feature
- Responsive design for mobile and desktop
- Custom T-shirt preview section
- User login and signup
- Order summary page

## Tech Stack
- HTML
- CSS
- JavaScript
- Bootstrap
- Node.js
- Express.js
- MongoDB

## Pages
- Home Page
- Product Listing Page
- Product Details Page
- Cart Page
- Wishlist Page
- Login / Signup Page
- Checkout Page
- Contact Page
